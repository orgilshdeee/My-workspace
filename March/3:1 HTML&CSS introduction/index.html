Орос-Украины уулзалтын талаар Зеленський: Бид хүссэн үр дүндээ хүрээгүй байна

Орос, Украины талууд Беларусийн хил орчимд бол таван цаг орчим уулзаад, хэлэлцээгээ өндөрлүүлжээ. Одоогоор уулзалтын үр дүнгийн талаар албан ёсны мэдээлэл өгөөгүй байна.

Тэд хоёрдугаар шатны уулзалт зохион байгуулах боломжийн талаар хэлэлцсэн ч тодорхой шийдвэрт хүрээгүй ажээ.
